## 🚀 Quick Start (3 Steps)

### Step 1: Start Dev Server (1 min)
```bash
cd /Users/ameygupta/infl/scroll-video-site
npm run dev
```
→ Visit http://localhost:3000

### Step 2: Add Your Video (5 min)
1. Prepare H.264 MP4 video (5-10 seconds)
2. Place at: `public/videos/janani-vikalakshi.mp4`
3. Update `backgroundColor` in `app/page.tsx`

### Step 3: Customize Content (5 min)
Edit `app/page.tsx`:
```jsx
const textOverlays = [
  { scrollProgress: 0.05, text: 'Your Headline', alignment: 'center' },
  // ... update with your content
];
```
